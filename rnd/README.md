# R&D (Research & Experimentation) Sandbox

This directory is dedicated to experimental features, prototypes, and research notes. 

## Guidelines
- Portions of code here may be unstable.
- Use `rnd/` branches for these features.
- Once a research item is validated, it should be refactored and moved into the main `engine/` or `agents/` directories.
